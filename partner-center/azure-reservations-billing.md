# <a name="microsoft-azure-reserved-vm-instances-billing"></a>Выставление счетов за услугу Microsoft Azure Reserved VM Instances

**Относится к**

-  Центр партнеров
-  Портал Microsoft Azure
-  Партнеры по программе CSP

Партнеры, участвующие в программе поставщиков облачных решений (CSP), теперь могут предлагать клиентам зарезервированные экземпляры на виртуальных машинах Microsoft Azure. Клиенты могут резервировать виртуальные машины заранее — на один год или три года — и получать значительную экономию при использовании Azure.   

Клиенты вносят предоплату за услугу Azure Reserved VM Instances. Приобретая услугу Azure Reserved VM Instances от лица клиента, вы получаете счета-фактуры и файлы выверки для этих единовременных платежей. 

>[!IMPORTANT]
>Если вы приобрели Azure Reserved VM Instances для клиента на рынке, где валюта отличается от вашей, для выставления счетов по умолчанию используется валюта рынка клиента, а не валюта вашего местоположения. Если у вас есть клиенты на нескольких рынках, вы будете получать отдельные счета и файлы выверки для каждой валюты клиентов, чтобы клиенты могли оплачивать счета в соответствующей валюте. 

Чтобы получить доступ к одноразовым счетам и файлам сверки с оплатой, выберите **выставление счетов** из центра партнеров, а затем выберите **один раз**. 

Другие общие сведения о выставлении счетов в рамках программы поставщиков облачных решений см. в разделе [Основы выставления счетов](billing-basics.md).

|DiscountDetails |Подробный список всех соответствующих скидок. |


## <a name="manage-your-billing"></a>Управление выставлением счетов

### <a name="view-your-current-billing-status-invoices-and-recon-files"></a>Вы можете посмотреть текущее состояние выставления счетов, счета-фактуры и файлы выверки

1.  В центре партнеров выберите **выставление счетов** и **один раз** , чтобы просмотреть состояние выставления счетов. 
2.  Выберите счет-фактуру или файл выверки, чтобы просмотреть более подробные сведения. 

### <a name="view-a-customers-order-history"></a>Просмотр журнала заказов клиента

1.  В меню центра партнеров выберите **Клиенты** .
2.  На странице **Клиенты** найдите клиента, чей журнал заказов вы хотите посмотреть, затем выберите стрелку вниз, чтобы развернуть запись клиента. 
3.  Выберите **Просмотр заказов**, чтобы открыть журнал заказов.

### <a name="create-a-credit-or-void-note"></a>Создание кредит-ноты или извещения об аннулировании

Иногда необходимо аннулировать счет и создать новый. Например, это может потребоваться, если у клиента изменилось название организации и клиент получил счет, в котором указано старое название. 

Чтобы аннулировать счет и создать новый, скачайте соответствующую форму в разделе корректировок на странице выставления счетов.
