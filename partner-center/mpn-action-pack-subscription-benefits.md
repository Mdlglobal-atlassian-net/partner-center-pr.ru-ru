# <a name="action-pack-subscription-benefits"></a>Активация преимуществ подписки Action Pack

В число преимуществ подписки Action Pack входят права на внутреннее использование новейшего программного обеспечения для поддержки работы компании, продажи решений, разработки приложений и обучения сотрудников. 

>**Важно.**<br>
Эти лицензии не могут перепродаваться или использоваться для прямого извлечения прибыли, в коммерческих и личных целях, а также для обучения клиентов.  

Основные преимущества теперь позволяют партнерам выбрать оптимальное сочетание интернет-служб и лицензий на локальное программное обеспечение. Партнеры могут выбрать решения для повышения производительности, демонстрации, разработки, тестирования и внутреннего обучения. 

Важные сведения о правах на использование лицензий и ограничениях для каждого программного продукта или интернет-службы см. в руководстве по использованию продукта. Ваша организация должна соблюдать условия использования, описанные в этом руководстве.
