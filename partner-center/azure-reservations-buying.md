# <a name="buy-microsoft-azure-reservations-on-behalf-of-your-customers-in-partner-center"></a>Купить резервирование Microsoft Azure для клиентов в центре партнеров 

**Применяется к**

-  Центр партнеров
-  Портал Microsoft Azure
-  Партнеры по программе CSP

## <a name="before-you-begin"></a>Перед началом работы

Просмотрите следующие важные сведения, прежде чем покупать резервирования Azure для клиентов.

-   Прежде чем приобретать резервирования от лица клиента, убедитесь, что у него уже есть активная подписка Azure
  
-   Затраты на программное обеспечение подписки, такие как программное обеспечение базы данных SQL или SUSE Linux не включены в цены Azure резервирования

-   Коммерческие цены корпорации Майкрософт для вас без учета налогов, если расположение не Бразилии. Если ваше расположение Бразилия, коммерческих цена вам включает соответствующие налоги 
 
-   Агентам по продажам и агентам службы технической поддержки требуется явный доступ к подписке Azure. Это позволяет агентам выполнять действия от лица клиента: совершать покупки и управлять подпиской на портале Azure, а также создавать запросы в службу поддержки, включая запросы на обмен и возмещение средств  

-   Если вы непрямой поставщик и покупаете резервирования Azure через портал Azure, зарегистрированный партнер (непрямой торговый посредник) наследуется от выбранной подписки Azure CSP. 

-   Изменить зарегистрированного партнера после приобретения резервирований Azure невозможно. Можно отменить существующее резервирование и приобрести новое от лица нового зарегистрированного партнера. 

-   Если клиент желает перенести подписку Azure из программы Direct или EA в программу CSP, резервирования не переносятся.

## <a name="purchase-azure-reservations"></a>Приобрести резервирование Azure

Выполните следующие действия, чтобы купить резервирование Microsoft Azure для клиентов в центре партнеров.

1. Выберите **клиентов** из меню центра партнеров.  

2. На странице **Клиенты** найдите клиента, желающего приобрести резервирования Azure, затем выберите стрелку вниз, чтобы развернуть запись клиента.  

3. Выберите **Добавить продукты**, а затем выберите **Azure**. 

    1. Выберите сегмент рынка клиента из **сегмент** списка.

    2. Выберите **резервирования** из продукта **тип** списка.

    В. Выберите тип резервирования, клиент хочет из **тип резервирования** списка.

4. Резервирования Azure должны быть связаны с активной подпиской Azure. Выберите подписки клиента, вы хотите добавить Azure резервирования из **подписки клиента** списка. 

   >[!IMPORTANT]
   >Если клиент еще нет активной подписки Azure, выберите **веб-служб** добавить один сейчас. 

5. Используйте фильтры для поиска Azure резервирования на виртуальных машинах, которые соответствуют требованиям клиента.  

6. Найдя резервирования, вы хотите приобрести, введите число зарезервированных экземпляров, клиенту потребуется в **количество** , а затем выберите **добавить в корзину**.  

7. Повторяйте шаги 5 и 6, пока в заказ не будут добавлены все необходимые позиции. Выберите **Проверка**, чтобы проверить правильность заказа.  

8. На странице **Проверка заказов** можно выполнять следующие действия. 

    - Проверить или изменить количество зарезервированных экземпляров.

    - Выбрать область резервирования. Область резервирования может охватывать одну или несколько подписок (общая область). Если вы задать область резервирования для одной подписки, скидка на резервирование применяется к только для этой подписки. Если выбрана общая область резервирования, скидка применяется к любым подпискам, оплачиваемым клиентом. 

      >[!NOTE] 
      >Если принято решение ограничить область резервирования для одной подписки Azure, может потребоваться увеличить квоту на виртуальные ЦП подписки. Чтобы увеличить квоту на виртуальные ЦП подписки, необходимо создать запрос в службу поддержки на портале Azure. Следуйте инструкциям [в этом разделе](https://docs.microsoft.com/azure/azure-supportability/resource-manager-core-quotas-request) для создания запроса.    

    - Если вы являетесь партнером-поставщиком, выберите торгового посредника, которого требуется связать с продуктом.

9. Выберите **Купить**, чтобы разместить заказ. Сведения о заказе, в том числе номер заказа, отображаются на **Подтверждение** страницы. Выберите **Готово**, чтобы перейти к странице **Журнал заказов**. 

10. Чтобы управлять в заказ на портале Azure, найдите клиента на вашей **клиентов** странице, а затем выберите стрелку вниз, чтобы развернуть строку клиента. Выберите **Портал управления Microsoft Azure**, чтобы открыть запись клиента на портале Azure.
