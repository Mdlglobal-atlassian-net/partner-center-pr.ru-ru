# <a name="use-the-reconciliation-files"></a>Использование файлов выверки

**Относится к**

-  Центр партнеров
-  Центр партнеров для Microsoft Cloud for US Government

**Соответствующие роли**

- Администратор выставления счетов
- Глобальный администратор

Для получения подробного представления по каждому элементу в цикле выставления счетов Скачайте файлы сверки из центра партнеров. Эти сведения включают плату за подписки каждого клиента и детализированные события (например, добавление количества мест к подписке в середине срока).

## <a name="formatting-issues"></a>Проблемы форматирования

Иногда файл разведывательную может иметь проблемы форматирования. (Это может произойти, например, если языковой стандарт EN-US не используется.) Чтобы устранить эти проблемы, выполните следующие действия. 

<ol>
<li>Откройте CSV-файл в Excel и выберите первый столбец. На ленте выберите <strong>данные</strong>, а затем выберите <strong>текст в столбцах</strong>.</li>

<li>В мастере преобразования текста в столбцы выберите <strong>Тип файла с разделителями</strong>, а затем нажмите кнопку <strong>Далее</strong>.</li> 

<li>В поле разделителями Выберите <strong>запятая</strong>. Если <strong>вкладка</strong> уже выбрана, можно оставить ее. Выберите <strong>Далее</strong>.</li>

<li>В поле Формат данных столбца выберите <strong>Date: mdy</strong>, а затем нажмите кнопку <strong>Далее</strong>.</li> 

<li>В поле Формат данных столбца выберите <strong>текст</strong> для всех столбцов суммы, а затем нажмите кнопку <strong>Готово</strong>.</li>
</ol>

## <a name="downloading-a-large-recon-file"></a>Скачивание большого файла разведывательную

Файлы разведывательную могут увеличиваться очень большими и иногда трудно загружать. Сведения о загрузке больших файлов разведывательную в скрипте PowerShell см. в разделе [Получение элементов строки счета](https://docs.microsoft.com/partner-center/develop/get-invoiceline-items).

## <a href="" id="charge_types"></a>Сопоставление расходов между счетом и файлом сверки

Ваш счет-фактура содержит сводку по расходам, а файл выверки включает подробную постатейную разбивку операций с указанием типов расходов.

Для добавления перекрестных ссылок на суммы расходов между счетом-фактурой и файлом выверки можно с помощью параметров фильтрования Microsoft Excel отфильтровать расходы по типам в файле выверки, чтобы сопоставить расходы по накладной с набором разбивок расходов в файле выверки.

Файлы выверки, как на основе лицензий, так и на основе использования, отражают только транзакции и расходы, связанные с использованием (израсходованные единицы и связанные с ними платежи). Один из кредитов, скидок или денег, которые отображаются в счете как "корректировки", не отображаются в файле сверки.
