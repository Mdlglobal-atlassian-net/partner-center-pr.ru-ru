# <a name="create-user-accounts-and-set-permissions"></a>Создание учетных записей пользователей и определение разрешений

**Относится к:**

-  Центр партнеров

Администратор создает учетную запись пользователя для каждого сотрудника партнера, которому необходим доступ к Центру партнеров. Эти задачи должны выполняться администратором с разрешениями **Глобальный администратор** или **Администратор управления пользователями**. В разделе **Параметры учетной записи** &gt; **Управление пользователем** можно добавлять учетные записи, а также определять и изменять разрешения.

**Добавление нового пользователя**

1.  В Центре партнеров последовательно выберите элементы "Информационная панель" &gt; **Параметры учетной записи** &gt; **Управление пользователем**.
2.  Нажмите кнопку **Добавить пользователя**.

3.  Введите полное имя пользователя и создайте уникальный адрес электронной почты для него.

4.  Выберите тип агента и тип администратора. Доступ к Центру партнеров основан на ролях. Поэтому варианты выбора в этом шаге определяют функциональные возможности, которые будут предлагаться пользователю в дальнейшем. Подробнее о возможностях каждой роли см. в разделе [Определение разрешений для пользователей](#setuserpermissions).

5.  Добавьте пользователя. Появится экран подтверждения с временным паролем для нового имени входа в систему. Вы должны скопировать этот пароль и отправить его новому пользователю — он недоступен после закрытия экрана. При первом входе в систему ему будет предложено обновить свой пароль.

### <a href="" id="setuserpermissions"></a>Определение разрешений для пользователей

Доступ к Центру партнеров основан на ролях. Это значит, что вы можете настроить представление пользователя на отображение только функций, которые будут необходимы пользователю для выполнения его должностных обязанностей. Для каждого пользователя необходимо выбрать два параметра:

-   Параметр **агент** управляет типом данных клиента и информации Майкрософт, которые может видеть пользователь.

-   Параметр **администратор** определяет, насколько пользователь контролирует среду Центра партнеров и все остальные службы Майкрософт (учетные записи, профили и запросы в службу поддержки). Этот параметр влияет на внешнюю среду Центра партнеров — администратор выставления счетов может получить доступ ко всем счетам за все службы Майкрософт (даже те, которые не связаны с CSP), а глобальный администратор может получить доступ к учетным записям пользователей и клиентов также за пределами CSP.

    По умолчанию значением параметра всегда должно быть **Не администратор**, если для роли пользователя не требуется дополнительный доступ для выполнения задач и поддержки пользователей.
