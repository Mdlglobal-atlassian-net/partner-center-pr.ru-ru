# <a name="azure-reserved-vm-instances-ri--server-subscriptions-for-azure"></a>Зарезервированные экземпляры виртуальных машин Azure (RI) + серверные подписки на Azure

**Область применения**

-  Центр партнеров
 
Корпорация Майкрософт [объявила](https://blogs.partner.microsoft.com/mpn/better-together-azure-reserved-instances-server-subscriptions/?ln=en-us) об открытии доступа к услуге Azure Reserved Virtual Machine Instances (RI). Кроме того, участникам программы поставщиков облачных решений (CSP) в ближайшее время станут доступны подписки на Windows Server и SQL Server. 

Корпорация Майкрософт включила Azure RI и серверные подписки в партнерскую программу CSP, чтобы удовлетворить быстро растущий спрос на экономичные решения для непрерывных и четко прогнозируемых облачных рабочих нагрузок. Программа CSP позволяет партнерам приобретать, развертывать и контролировать Azure RI и серверные подписки от имени коммерческих клиентов через Центр партнеров Майкрософт и портал Azure.  
 
### <a name="compelling-new-azure-offer-in-csp"></a>Новое интересное предложение Azure в программе CSP 

Виртуальные машины Azure предоставляют клиентам гибкие возможности виртуализации для различных вычислительных решений, включая разработку и тестирование, запуск приложений и расширение центра обработки данных.  

Благодаря услуге Reserved VM Instances, предоставляемой в рамках программы CSP, коммерческие клиенты могут сэкономить до 72% по сравнению с оплатой виртуальных машин Azure по мере использования. Для этого достаточно приобрести (зарезервировать) виртуальную машину на срок от 1 года до 3 лет. Клиенты с Windows Server, которые уже успели оценить преимущества гибридного использования Azure и поддержки Software Assurance, могут сэкономить до 80% по сравнению с оплатой по мере использования.  

Корпорация Майкрософт дополнила эти замечательные предложения, обеспечив удобство выбора и невероятно гибкие условия развертывания. Преимущества Azure Reserved VM Instances:  

-   Эта услуга предоставляется для "базовой виртуальной машины", поэтому клиенту не нужно заранее принимать решение о развертывании виртуальной машины с Windows Server или другим продуктом. Просто укажите регион Azure, тип виртуальной машины и срок — 1 год или 3 года. 

-   Резервирования можно связать с несколькими подписками в учетной записи клиента или с одной конкретной подпиской.  

-   Можно в любое время изменить экземпляры, семейство, местоположение и срок.  

-   Можно отказаться от услуги в любое время в течение срока резервирования.  

-   Оцените приоритетное использование вычислительной мощности в регионах Azure 
 
Подписки на Windows Server и SQL Server, продаваемые в рамках программы CSP, также включают в себя преимущества гибридного использования Azure, которые ранее были доступны только в программе Software Assurance с традиционными программами лицензирования. Преимущества гибридного использования Azure обеспечивают значительную экономию по сравнению с оплатой по мере использования. Клиенты партнеров по программе CSP, резервирующие вычислительные мощности Azure (через RI) и серверное программное обеспечение Майкрософт (по подписке) на 1 год или 3 года, могут получить максимальную экономию при использовании облачных вычислений. Это преимущество особенно заинтересует клиентов, работающих только с партнерами CSP и не имеющих доступа к программе поддержки Software Assurance. 

Благодаря уникальному сочетанию привлекательных цен и гибкости развертывания клиенты, выбравшие Azure Reserved VM Instances и серверные подписки, получат максимальное совокупное преимущество. 
 
### <a name="why-is-this-announcement-good-news-for-microsoft-partners"></a>Почему это объявление важно для партнеров Майкрософт? 

Корпорация Майкрософт включила Azure Reserved VM Instances и серверные подписки с преимуществами гибридного использования Azure в партнерскую программу CSP, чтобы помочь партнерам в развитии облачных решений. Партнеры могут применять экономичные виртуальные машины Azure и серверное программное обеспечение, которые оптимизированы для более интенсивных облачных рабочих нагрузок. До недавнего времени партнеры CSP могли продавать только виртуальные машины Azure с оплатой по мере использования, и им приходилось применять другие программы лицензирования для приобретения подписок на программное обеспечение Windows Server и SQL Server.  

Удобная бизнес-модель лицензирования Azure Reserved VM Instances и серверных подписок на уровне клиента позволяет партнерам прогнозировать расходы, улучшает масштабируемость и упрощает процесс продажи. Партнерам не нужно внедрять сложные процессы и инструменты для приобретения, управления и расчета стоимости зарезервированных экземпляров для различных клиентов. Благодаря этому партнеры могут уделять больше времени потребностям клиентов и расширять свой портфель дополнительных облачных решений для управления ИТ-инфраструктурой. 

Услуга Azure Reserved VM Instances и серверные подписки представляют интерес для партнеров, поскольку служат в качестве надежной платформы для увеличения доходов и рентабельности. Важнее всего то, что облачные службы Azure позволяют партнерам разрабатывать и предоставлять прибыльные ИТ-сервисы премиум-класса, стимулирующие продажи дополнительного программного обеспечения и/или подписок и позволяющие взимать постоянную плату за услуги управления. Во-вторых, мы предлагаем партнерам передовую экономическую модель, которая позволяет получить максимальную прибыль от облачных решений. И наконец, следует отметить, что решения на основе подобных обязательств способствуют построению надежных и доверительных деловых отношений с клиентами, обеспечивающих более высокую прибыль в долгосрочной перспективе.  

Таким образом, Azure Reserved VM Instances и серверные подписки в рамках программы CSP обеспечивают наибольшую выгоду для клиентов по сравнению с другими аналогичными предложениями. Благодаря простой и удобной коммерческой модели партнеры получают надежную платформу для увеличения прибыли и могут использовать больше возможностей облачных вычислений.

## <a name="azure-hybrid-benefit"></a>Преимущество гибридного использования Azure
[Преимущество гибридного использования Azure](https://azure.microsoft.com/en-us/pricing/hybrid-benefit) помогает получить дополнительную прибыль от лицензий на Windows Server и сэкономить до 40 процентов* на виртуальных машинах. Это преимущество можно использовать с лицензиями Windows Server Datacenter и Standard edition, зарегистрированными в программе Software Assurance. В зависимости от выпуска можно преобразовывать и повторно использовать лицензии для работы с виртуальными машинами Windows Server в Azure и оплачивать использование вычислительной мощности по более низким тарифам (тарифы для виртуальных машин Linux).

Преимущество гибридного использования Azure — самое экономичное облако для миграции Windows Server или SQL Server:
-   Сэкономьте до 82% на Windows Server с преимуществом гибридного использования Azure и зарезервированными экземплярами
-   Сэкономьте до 30% * при миграции на Azure SQL Database2 с преимуществом гибридного использования Azure
-   Работайте в своем темпе — переносите отдельные рабочие нагрузки или целые центры обработки данных
-   Обеспечьте максимальную эффективность текущих инвестиций в программу Software Assurance

См. также [Преимущество гибридного использования Azure — вопросы и ответы](https://azure.microsoft.com/en-us/pricing/hybrid-benefit/faq/)

*Фактическая экономия может различаться в зависимости от региона, типа экземпляра и использования
