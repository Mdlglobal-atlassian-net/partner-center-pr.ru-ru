# <a name="assign-users-roles-and-permissions"></a>Назначение пользователям ролей и разрешений

**Область применения**

- Центр партнеров

Каждая роль в центре партнеров поставляется вместе с определенные разрешения, позволяя пользователю для выполнения определенных задач. Всем, кто работает в центре партнеров необходимо назначить роли.

В зависимости от размера организации некоторые пользователи могут работать на разных расположений или только для одного местоположения. 

Пример.

- Глобальный администратор может быть администратором официальное и указанного местоположения; глобальный администратор может быть администратора только для одного из расположений, находящейся или. 
- Администраторами программ поощрений могут администрировать для одного или нескольких расположениях, но необходимо указать эти расположения.


В следующей таблице описаны возможности каждой роли в центре партнеров. При планировании назначения ролей проанализируйте, что тот или иной пользователь делает в вашей организации. Например, вы можете назначить глобального администратора агентом администратора. Пользователь, который занимается маркетингом, также может управлять бизнес-профилем.
