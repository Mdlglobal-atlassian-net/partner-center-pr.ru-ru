# <a name="usage-based-file-fields"></a>Поля файла с учетом использования

Область применения

- Центр партнеров
- Центр партнеров для Microsoft Cloud for US Government

Чтобы согласовать расходы по использованию клиента, Сравните **реселлерид**, **ResellerName**и **реселлербиллаблеаккаунт** из файла сверки с **именем клиента** и **идентификатором подписки** из центра партнеров.

## <a name="fields-in-usage-based-reconciliation-files"></a>Поля в файлах сверки с учетом использования

В следующих полях объясняется, какие услуги были использованы и по какому тарифу.

| Column | Описание | Выборка значений |
| ------ | ----------- | ------------ |
| PartnerId | Идентификатор партнера в формате GUID. | *DA41BC5F-C52D-4464-8A8D-8C8DCC43503B* |
| PartnerName | Имя партнера. | *Contoso, Ltd.* |
| партнербиллаблеаккаунтид | Идентификатор учетной записи партнера. | *1010578050* |
| кустомеркомпанинаме | Название организации клиента, зарегистрированное в центре партнеров. *Очень важно для согласования счета со сведениями о системе.* | *Тестирование клиента* |
| MpnId | Идентификатор MPN партнера CSP. | *4390934* |
| реселлермпнид | MPN Идентификатор торгового посредника записи для подписки.  |
| InvoiceNumber | Номер счета-фактуры, содержащего указанную транзакцию. | *D020001IVK* |
| ChargeStartDate | Дата начала цикла выставления счетов, за исключением случаев представления дат ранее неоплаченного скрытого использования данных (из предыдущего цикла выставления счетов). Временем всегда является начало дня, 0:00. | *2/1/2019 0:00* |
| ChargeEndDate | Дата окончания цикла выставления счетов, за исключением случаев, когда представлены даты ранее неоплаченного скрытого использования данных (из предыдущего цикла выставления счетов). Временем всегда является конец дня, 23:59. | *2/28/2019 23:59* |
| SubscriptionId | Уникальный идентификатор подписки в платформе выставления счетов Майкрософт. При обращении в службу поддержки может оказаться полезным указать подписку. Не используется для сверки. *Это значение не совпадает с **идентификатором подписки** в консоли администратора партнера.* | *ускбмгааааааааиа* |
| SubscriptionName | Псевдоним для предложения услуги. | *Microsoft Azure* |
| SubscriptionDescription | Предложение службы бизнес-приложения. | *Microsoft Azure* |
| OrderID | Уникальный идентификатор заказа на платформе выставления счетов Майкрософт. При обращении в службу поддержки может оказаться полезным указать подписку. Не используется для сверки. | *566890604832738111* |
| ServiceName | Имя соответствующей службы Azure. | *ВИРТУАЛЬНЫЕ МАШИНЫ* |
| ServiceType | Конкретный тип службы Azure. | *Служебная шина — индивидуальная или Pack*, *SQL Azureная база данных — Business или Web Edition* |
| ResourceGuid | Определенный уникальный идентификатор для всех данных и ценовой структуры службы. | *DA41BC5F-C52D-4464-8A8D-8C8DCC43503B* |
| ResourceName | Имя ресурса Azure. | *Передача данных в (ГБ)* , *Передача исходящих данных (ГБ)* |
| Region | Регион, к которому применяется использование. В основном используется для назначения ставок для передачи данных, так как ставки зависят от региона. | *Азиатско-Тихоокеанский регион*, *Европа*, *Латинская Америка*, *Северная Америка* |
| SKU | Уникальный идентификатор Майкрософт для предложения. | *7UD — 00001* |
| DetailLineItemId | Идентификатор и количество для детализации различных ставок для службы или ресурса в заданном расчетном периоде. Для многоуровневого ценообразования Azure может существовать одна ставка за определенное количество оплачиваемых единиц, а затем другой тариф. | *1* |
| ConsumedQuantity | Объем потребляемой службы (например, часы или ГБ) за отчетный период. Также включает использованный объем за предыдущие отчетные периоды, по которому не выставлялся счет. | *11* |
| IncludedQuantity | Единицы в составе предложения. Обычно отсутствует в CSP. | *0* |
| OverageQuantity | Единицы, не входящие в состав предложения. Они должны оплачиваться партнером. Равно **консумедкуантити** минус **инклудедкуантити**. | *11* |
| ListPrice | Цена предложения действует на дату начала подписки. | *$0,0808* |
| PretaxCharges | Равно **листприст** , умноженному на **OverageQuantity**, округленное до ближайшего цента. | *$0,085* |
| TaxAmount | Сумма налога с оплатой. В зависимости от правил налогов на рынке и определенных обстоятельств. | *$0,08* |
| PostTaxTotal | Итоговая сумма за вычетом налога, если таковой применим. | *$0,93* |
| Валюта | Тип валюты. Каждый объект выставления счетов имеет только одну валюту. Убедитесь, что он соответствует первому счету, а затем после всех основных обновлений платформы выставления счетов. | *1000,00* |
| PretaxEffectiveRate | Цена за единицу до вычета налогов. Равно **PretaxCharges** , разделенному на **OverageQuantity**, округленное до ближайшего цента. | *$0,08* |
| PostTaxEffectiveRate | Цена за единицу после вычета налога. Равно **PostTaxTotal** , разделенному на **OverageQuantity**, округленное до ближайшего цента. Или равно **претаксеффективерате** плюс групповой ставка налога на единицу амаун, округленную до ближайшего цента. | *$0,08* |
| ChargeType | [Тип оплаты](recon-file-charge-types.md) или корректировки. | См. раздел [типы начислений](recon-file-charge-types.md). |
| CustomerId | Уникальный идентификатор Майкрософт для клиента в формате GUID. | *ORDDC52E52FDEF405786F0642DD0108BE4* |
| DomainName | Доменное имя клиента. Это поле может быть пустым до второго цикла выставления счетов. | *example.onmicrosoft.com* |
| BillingCycleType | Частота выставления счетов по времени.| **Ежемесячно**  |
| Unit | Единица **имени**ресурса. | *ГБ* или *часы* |
| CustomerBillableAccount | Уникальный идентификатор учетной записи на платформе выставления счетов Майкрософт. | *1280018095* |
| UsageDate | Дата развертывания службы. | *2/1/2019 0:00* |
| MeteredRegion | Определяет расположение центра обработки данных в регионе (для служб, где это значение применимо и заполняется). | *Восточная Азия*, *Южная Восточная Азия*, *Северная Европа*, *Западная Европа*, *северо-центральная часть США*, Юго-Центральный *регион* США |
| Измеренная_служба | Определяет индивидуальные сведения об использовании службы Azure, если они не определены в столбце **ServiceName** . Например, передача данных отображается как *Microsoft Azure — все службы* в столбце **ServiceName** . | *AccessControl*, *CDN*, среда выполнения *вычислений*, *база данных*, *servicebus*, *хранилище* |
| MeteredServiceType | Подзаголовок для поля **метередсервице** , в котором содержится дополнительное разъяснение использования службы Azure. | *EXTERNAL* |
| Project (Проект) | Определенное клиентом имя для экземпляра службы. | *ORDDC52E52FDEF405786F0642DD0108BE4* |
| ServiceInfo | Количество подключений к служебной шине Azure, которые были подготовлены и использованы в указанный день. | *1,000000 подключений в течение 30 дней* (если в течение 30-дневного месяца было настроено отдельное подключение), *25 подключений/30 дней — используется: 1,000000* (если подготовлено подключений служебной шины 25 и в этот день используется 1) |
