# <a name="sell-software-subscriptions-through-csp"></a>Продажа подписок на программное обеспечение через программу CSP

**Относится к**

- Поставщики облачных решений

**Соответствующие роли**

- Агент администрирования
- Глобальный администратор

Благодаря резервированию Azure и серверным подпискам (Windows Server и SQL Server Subscriptions) партнеры в программе CSP могут более эффективно обращаться к быстро растущему запросу клиентов, чтобы обеспечить более экономичное и устойчивое решение для обеспечения высокой прогнозируемости. облачные рабочие нагрузки. 

Теперь вы можете получать, подготавливать и администрировать резервирование и серверные подписки Azure от имени коммерческих клиентов через центр партнеров и портал Azure, используя преимущества Преимущество гибридного использования Azure.

Преимущество гибридного использования Azure помогает получить дополнительную прибыль от лицензий на Windows Server и сэкономить до 40 процентов на виртуальных машинах. Это преимущество можно использовать с лицензиями Windows Server Datacenter и Standard Edition, зарегистрированными в программе Software Assurance. В зависимости от выпуска вы можете преобразовать или повторно использовать лицензии для запуска виртуальных машин Windows Server в Azure и оплатить меньшую базовую частоту вычислений (тарифы виртуальных машин Linux, например).

## <a name="buy-software-subscriptions-on-behalf-of-customers"></a>Приобретение подписок на программное обеспечение от имени клиентов

Чтобы приобрести подписки на программное обеспечение от имени клиента, выполните следующие действия.

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard)центра партнеров.

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. На странице сведений о клиенте выберите **Добавить продукты**, а затем следуйте инструкциям на экране, чтобы создать заказ и заплатить за него. Все коммерческие цены исключают налог за исключением Австралии и Бразилии. Для Австралии и Бразилии Цена включает налог.

## <a name="activate-and-manage-software-subscriptions"></a>Активация подписок на программное обеспечение и управление ими

После приобретения подписки на программное обеспечение выполните следующие действия, чтобы скачать ее.

>[!NOTE]
>Для загрузки программного обеспечения и получения ключей активации необходимо быть агентом администратора.

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard)центра партнеров.

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. Перейдите на страницу сведений о клиенте и выберите **Программное обеспечение**. Вы увидите список всех программных продуктов, приобретенных от имени клиента. 

4. Разверните продукт, который необходимо скачать. В поле **Выбор продукта** выберите необходимые **Версию**, **Язык** и **Тип файла/ОС**. 

5. Выберите **Отправить**, чтобы отобразить конкретные продукты. 

6. Выберите **Получить ключи и загружаемые файлы**. 

7. Выберите **Скачать**, чтобы начать скачивание, или выберите **Копировать ссылку**, чтобы скопировать ссылку и отправить ее клиенту. 

>[!NOTE]
>Срок действия этой ссылке истечет через две недели или после выполнения 50 скачиваний в зависимости от того, какое из этих событий произойдет первым. После истечения срока действия ссылки вернитесь на эту страницу и выберите пункт **Получить ключи и загружаемые файлы** еще раз, чтобы продлить срок действия ссылки еще на две недели или получить 50 дополнительных скачиваний. Это можно сделать столько раз, сколько необходимо. 

## <a name="view-activity-for-software-key-access-and-software-downloads"></a>Просмотр действий по доступу к ключам программного обеспечения и загрузке программного обеспечения

В целях аудита или соответствия может потребоваться проверить список пользователей, имеющих доступ к программным ключам подписки сервера или загруженному программному обеспечению серверной подписки. Для доступа к этим сведениям используйте следующую процедуру. 

>[!NOTE]
>Для просмотра этих журналов действий необходимо быть глобальным администратором, администратором учетной записи, администратором ссылок или администратором маркетингового содержимого. 

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard)центра партнеров.

2. Щелкните значок шестеренки в правом верхнем углу.

3. В меню выберите **Журнал действий**.

4. Введите диапазон дат для действия, которое вы хотите просмотреть. В журнале действий отобразится список пользователей, имеющих доступ к программным ключам или загруженному программному обеспечению в указанное время. 

## <a name="cancel-a-purchase"></a>Отмена покупки

Вы можете отменить покупку программного обеспечения в течение 60 дней с даты покупки. Если вы отменили этот период в течение первого 60-дневного периода, вы не будете взимать плату за ранний увольнение. Через 60 дней вы больше не сможете отменить покупку. (См. Примечание по ключевым ограничениям для этого правила отмены. Дополнительные сведения о том, что происходит после отмены покупки программного обеспечения, см. в разделе также важное примечание после выполнения этих действий.) 

>[!NOTE]
>Следующие шаги по отмене покупки применяются только к программному обеспечению, которое подходит для отмены в конкретном окне отмены, например в течение первых 60 дней после покупки. Обратите внимание, что эти действия по отмене покупки не относятся к покупкам расширенных обновлений для системы безопасности Microsoft Windows 7 2020. (Чтобы отменить Windows 7 ЕСУ 2020, необходимо отправить запрос в службу поддержки клиентов, как описано далее в этом разделе.) Эти действия также не применяются к плану программного обеспечения SUSE Linux или RedHat в Azure. В настоящее время вы не можете отменять или обмениваться планом по SUSE или RedHat. [Узнайте больше](https://docs.microsoft.com/azure/virtual-machines/linux/prepay-suse-software-charges) об использовании планов SUSE Linux или RedHat.

Выполните следующие действия, чтобы отменить покупку.

>[!NOTE]
>Необходимо отменить покупку агента администратора. Ниже описана процедура отмены покупки на панели мониторинга центра партнеров. Это также можно сделать с помощью [API центра партнеров](https://docs.microsoft.com/partner-center/develop/cancel-software-purchases).

1. Прежде чем начать процесс отмены, убедитесь, что у вас есть следующее:

    - Имя клиента, GUID клиента или доменное имя

    - Имя продукта, который требуется отменить
    
    - Идентификатор заказа

2. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard)центра партнеров.

3. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

4. На странице сведения о клиенте выберите **программное обеспечение** , чтобы просмотреть список программного обеспечения, приобретенного для клиента. 

5. Найдите приобретенное программное обеспечение, которое нужно отменить, и нажмите **кнопку Отмена**. Появится диалоговое окно.

6. В раскрывающемся списке номер заказа выберите правильный идентификатор заказа, который нужно отменить, а затем нажмите кнопку **Отправить**.

7. Если вы хотите отменить несколько заказов для клиента, необходимо выполнить шаги с 4 по 6 для каждого уникального идентификатора заказа. (Дополнительные сведения о заказе или ИДЕНТИФИКАТОРе заказа см. на странице « **История заказов** клиента».)

8. При попытке отменить заказ центр партнеров также может получить другие сведения (которые отображаются под раскрывающимся списком номер заказа). Эти сведения могут включать:

    - Сколько дней осталось для отмены в определенном заказе

    - Было ли вы уже прошли окно отмены и больше не можете отменять заказ

    - Если нам нужны дополнительные сведения о запросе на отмену, вам может быть предоставлена ссылка на форму **запроса на поддержку клиентов** .

>[!IMPORTANT]
>После отмены заказа появится сообщение, подтверждающее вашу отмену. Однако может возникнуть задержка до 15 минут, прежде чем отмена появится на панели мониторинга центра партнеров. Также обратите внимание на следующие сведения после отмены.

После отмены покупки:

- Все связанные ключи программного обеспечения и ссылки для скачивания будут отозваны. Это означает, что ни вы, ни ваш клиент не можете использовать ключи программного обеспечения и ссылки для скачивания, которые больше не связаны с этой покупкой. Вы и ваш клиент несете ответственность за прекращение использования всех отмененных программ. Вы также несете ответственность за удаление отмененного программного обеспечения и снятие всех связанных с ним обновлений и ссылок.

- Отмененный элемент по-прежнему будет отображаться на странице сведений о программном обеспечении клиента, но ключ активации будет недоступен.

- В следующем ежемесячном счете будет отображаться кредит для отмененного заказа. Бессрочное программное обеспечение получит 100% кредита, а подписки на программное обеспечение получат пропорциональное кредитовое возмещение.

### <a name="submit-a-customer-support-request-to-cancel-a-purchase"></a>Отправьте запрос в службу поддержки клиентов, чтобы отменить покупку

Если вы попытались отменить покупку программного обеспечения через центр партнеров, но указали дополнительные сведения и заполнили форму запроса на поддержку клиентов, эти действия могут помочь вам:

1. При выборе ссылки **запросить службу поддержки** в окне Отмена покупки откроется страница сообщить о проблеме со службой центра партнеров.

2. В разделе сведения в списке Тип проблемы выберите **поставщик CSP покупки/возмещение от имени клиентов**.

3. Заполните поля влияние и название.

4. В поле Описание укажите следующие сведения:

    - Код GUID клиента пользователя или доменное имя
    
    - Идентификатор заказа или подписки
    
    - Причина возврата денежных средств

    - Запрошенная сумма

5. В поле Contact (контакт) введите свое имя, адрес электронной почты и номер телефона.

6. Если по какой-либо причине требуется приложить файл, выберите **Добавить файлы**. Этот шаг является необязательным.

7. По завершении нажмите кнопку **Отправить**.
