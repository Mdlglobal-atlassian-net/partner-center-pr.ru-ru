# <a name="partner-center-for-microsoft-cloud-germany"></a>Центр партнеров Microsoft Cloud в Германии

**Относится к:**

-  Центр партнеров Microsoft Cloud в Германии

## <a name="microsoft-cloud-germany-meeting-german-data-protection-requirements"></a>Microsoft Cloud в Германии: соблюдение немецких требований к защите данных 

Microsoft Cloud в Германии соответствует строгим немецким требованиям в отношении защиты данных и подразумевает наличие немецкого доверенного лица, управляющего физическим и логическим доступом к данным клиентов в соответствии с законодательством Германии. Все данные пользователей и системы, обрабатывающие данные пользователей или управляющие доступом к ним, находятся в центрах обработки данных в Германии. Сеть между этими центрами обработки данных изолирована в Германии.

Microsoft Cloud в Германии предоставляет отдельные экземпляры [Microsoft Azure](https://go.microsoft.com/fwlink/?linkid=847992), Office 365 и Microsoft Dynamics, размещенные в центрах обработки данных в Германии. В результате коммерческие пользователи из Европейского Союза и Европейской ассоциации свободной торговли (ЕАСТ) могут хранить данные клиентов и управлять ими в соответствии с законодательством Германии, а также ключевыми международными стандартами. Сотрудники корпорации Майкрософт не имеют доступа к данным клиентов без одобрения и контроля со стороны клиентов или немецкого доверенного лица.

Дополнительные сведения см. в разделе [Техническое описание Microsoft Cloud в Германии](http://download.microsoft.com/download/6/1/3/613C9ECB-9167-4EF5-B131-3BAD8D8A126C/Microsoft_Cloud_Germany_Datasheet.pdf).

## <a name="partner-center-for-microsoft-cloud-germany-selling-german-cloud-solutions-in-eu-and-efta"></a>Центр партнеров Microsoft Cloud в Германии: продажа немецких облачных решений в ЕС и ЕАСТ

После регистрации в программе поставщиков облачных решений (CSP) для Microsoft Cloud в Германии, вы будете использовать Центр партнеров для продажи немецких облачных решений и услуг пользователям в странах и регионах ЕС и ЕАСТ. 

>**Примечание.**<br>
На момент публикации этой статьи следующие функции Центра партнеров были недоступны в Центре партнеров Microsoft Cloud в Германии (с момента публикации некоторые функции могли быть добавлены в Центр партнеров, и этот список мог устареть).

- Поиск партнера
- Бизнес-профили
- Рекомендации
- Аналитика лицензий
- Данные клиента
- Центр уведомлений
- Модель и параметры косвенных торговых посредников/поставщиков
- Проверка работоспособности службы
- Регистрация независимых поставщиков программного обеспечения (ISV)
- Специальные требования клиентов

В соответствии с законодательством Германии в отношении сбора и использования персональных данных, а также ключевыми международными стандартами следующие задачи Центра партнеров выполняются с помощью отдельных порталов обслуживания.

## <a name="frequently-asked-questions-about-the-csp-program-for-microsoft-cloud-germany"></a>Вопросы и ответы по программе CSP для Microsoft Cloud в Германии

Этот раздел содержит ответы на часто задаваемые вопросы о программе поставщиков облачных решений (CSP) для Microsoft Cloud в Германии. Для получения дополнительных сведений вы также можете обратиться к своему менеджеру по работе с клиентами. 

**Что такое платформа Microsoft Cloud в Германии?** 

Платформа Microsoft Cloud в Германии объединяет в себе экземпляр лидирующих в отрасли облачных служб Майкрософт, уникальную архитектуру управления системами, а также специальную сеть, расположенную на территории Германии. В рамках этой уникальной модели, Доверенное лицо, работающее с данными— независимая немецкая компания— управляет доступом к данным клиентов и не передает их третьим лицам, за исключением случаев, когда клиент дает на это свое разрешение или этого требует немецкое законодательство. Эта модель также называется суверенитетом данных. 

**Кто имеет право принимать участие в программе CSP для Microsoft Cloud в Германии?**

Партнеры в странах Европейского Союза (ЕС), в Европейской ассоциации свободной торговли (EFTA) и в Великобритании могут принимать участие в программе CSP для Microsoft Cloud в Германии и продавать продукты/службы клиентам из стран Европейского Союза, Европейской ассоциации свободной торговли, а также в Великобритании. 

**Коммерческая программа CSP Commercial— это существующая партнерская программа. Чем от нее отличается программа CSP для Microsoft Cloud в Германии?**

Программа CSP для Microsoft Cloud в Германии отличается от коммерческой программы CSP Commercial по следующим аспектам. 
-   Право на участие. Программа CSP для Microsoft Cloud в Германии доступна только партнерам в ЕС, ЕАСТ и Великобритании. 
-   Регистрация. Регистрация партнеров в программе CSP для Microsoft Cloud в Германии должна осуществляться отдельно от регистрации в коммерческой программе CSP Commercial. 
-   Лицензирование. Лицензионная документация по программе CSP для Microsoft Cloud в Германии обновляется в соответствии с особенностями роли Доверенного лица. Дополнительные сведения об изменениях в лицензировании указаны ниже. 
-   Работа Центра партнеров. Чтобы обеспечить соответствие требованиям, предъявляемым к суверенитету данных, некоторые возможности Центра партнеров будут недоступны для программы CSP для Microsoft Cloud в Германии. Вместо этого партнеры могут использовать портал Microsoft Online Portal (MOP), службу Azure Active Directory (Azure AD) или набор веб-служб API Graph. 

**Чем лицензирование в рамках программы CSP для Microsoft Cloud в Германии отличается от лицензирования в рамках коммерческой программы CSP Commercial?**

Соглашение торгового посредника Microsoft Cloud (MCRA) и дистрибьюторское соглашение Microsoft Cloud (MCDA) для программы CSP для Microsoft Cloud в Германии отличаются от соответствующих соглашений для коммерческой программы CSP Commercial по следующим двум аспектам. 
-   Согласно требованиям этих соглашений, клиенты должны получить соглашение Microsoft Cloud (MCA) для облачной платформы в Германии вместе с Дополнением к Условиям использования онлайн-служб для облачной платформы в Германии 
-   Они включают в себя два изменения требований к данным клиента в соглашении MCA для облачной программы в Германии по сравнению с коммерческим соглашением MCA: 
-   Оно содержит Дополнение к Условиям использования онлайн-служб, определяющее дополнительные обязательства, связанные с обеспечением безопасности данных, которые корпорация Майкрософт несет по контролю местоположения данных, деятельности Доверенного лица и доступа к данным клиентов. 
-   Оно содержит Соглашение с доверенным лицом, работающим с данными. Приложение 1 к Дополнению содержит Соглашение с доверенным лицом, согласно которому компания T-Systems связывает себя обязательствами непосредственно с клиентом в отношении контроля данных клиента и управления доступом к ним 

**Каковы критерии допуска к осуществлению продаж в рамках программы CSP для Microsoft Cloud в Германии?**

Критерии допуска к осуществлению продаж в рамках программы CSP для Microsoft Cloud в Германии аналогичны критерием допуска к осуществлению продаж в рамках коммерческой программы CSP Commercial. Этим критериям соответствуют как прямые партнеры, так и косвенные торговые посредники. Если вы утверждены в качестве косвенного торгового посредника в рамках коммерческой программы CSP Commercial, необходимость в повторном прохождении процедуры утверждения в качестве косвенного торгового посредника в рамках программы CSP для Microsoft Cloud в Германии отсутствует.

Нет. Поощрения для партнеров в рамках программы CSP для Microsoft Cloud в Германии аналогичны поощрениям для партнеров в рамках коммерческой программы CSP Commercial. Дополнительные сведения о поощрениях для партнеров см. на портале MPN.

**Для каких клиентских сегментов можно осуществлять продажи?**

После утверждения вас в качестве партнера по программе CSP и заключения дистрибьюторского соглашения Microsoft Cloud вы сможете продавать продукты/службы любым коммерческим клиентам в странах ЕС, ЕАСТ и в Великобритании. 

**Какие продукты/службы будут доступны для продажи в рамках программы CSP для Microsoft Cloud в Германии?**

На момент запуска программы для продажи будут доступны продукты MicrosoftOffice365Germany и MicrosoftAzureGermany. Продукт MicrosoftDynamics365Germany (CRMOL) станет доступен во втором квартале 2017года. Мы будем добавлять в этот список другие службы по мере их выхода. 

**Доступны ли все службы Azure в рамках программы CSP для Microsoft Cloud в Германии?**

В настоящий момент доступны следующие предложения: «инфраструктура как услуга» (IaaS), базовая платформа и решения для работы с данными. План добавления продуктов регулярно обновляется и мы предоставим соответствующую документацию по нему, аналогичную документации для коммерческой программы CSP Commercial, в Центре партнеров (требуется проверка подлинности).
